Delimited Continuation Parser (DeCaP)

DeCaP is a new parser combinator library written in OCaml developped by Rodolphe Lepigre and Christophe Raffalli at the LAMA (LAboratoire de MAthématiques de l'Université Savoie Mont Blanc, FRANCE). Although DeCaP can be used to write parsers directly, it is intended to be used in conjunction with pa_ocaml, which is a full-featured parser for the OCaml language, written using DeCaP. These two tools together provide an easy and integrated way to write parsers and syntax extensions for the OCaml language.

List of the components:

  • Input: a module implementing an input buffer as a lazy stream of lines
  • Charset: a module implementing efficien character sets
  • Decap: the main module containing the parser combinators
  • pa_ocaml: full-featured ocaml parser
  • pa_parser: syntax extension for writing parsers in a BNF-like format

Main features:

  • Blank functions
  • Quotation / antiquotation mechanism (similar to camlp4)
